# orocommerce-report-plugin
This extension provides six predefined, highly useful reports to help you gain deeper insights into your sales and customers.

## Requirements

| | Version       |
| :--- |:--------------|
| PHP  | 8.2, 8.3, 8.4 |
| OroCommerce | 5.0, 5.1, 6.0 |
